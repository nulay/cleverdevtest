# cleverdevtest-client-service

REST API server to get client notes

---

## Getting Started

# Startup

This application is based on Spring Boot.

Requirements:

- JDK 15
- Maven
- Application starts with command

```
    mvn spring-boot:run
```

# API

- Default port 8081
- Endpoints are in SwaggerUI: http://localhost:8081/swagger-ui
